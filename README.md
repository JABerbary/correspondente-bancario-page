# correspondente-bancario-page

## Como executar o projeto

### 1. Clonar o repositório

```bash
git clone URL_DO_REPOSITORIO
```

---

### 2. Abrar o projeto no VS Code

Entre na pasta do projeto:

```bash
cd correspondente-bancario-page
```

Depois abra no VS Code:

```bash
code .
```

---

### 3. Instalar a extensão Live Server

No VS Code:

1. Clique em **Extensions** no menu lateral
2. Pesquise por:

```txt
Live Server
```

3. Instale a extensão criada por **Ritwick Dey**

Ou acesse diretamente:

[Live Server - VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer&utm_source=chatgpt.com)

---

### 4. Executar o projeto

1. Abra o arquivo:

```txt
index.html
```

2. Clique com o botão direito no arquivo

3. Selecione:

```txt
Open with Live Server
```

O projeto será aberto automaticamente no navegador.

---
